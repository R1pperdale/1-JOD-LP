Array de Objetos

const pessoas = [
{nome:"Maria",idade:30,cidade:"São Paulo"}
{nome:"João",idade:22,cidade:"Santos"}
{nome:"José",idade:38,cidade:"São Caetano do Sul"}
]

Neste exemplo o array tem o nome de pessoas e possui 3 objetos;
Cada objeto representa uma pessoa e cada um deles possui 3 propriedade: nome, idade e cidade.

Levando em conta os métodos já desenvolvidos, segue a lista de exercícios.

Crie uma Arrow Function que receba um array de 8 números variados e que retorne um novo array em cada número é classificado como "par" ou "ímpar", dependendo do valor. (Utilizar o método map)
Crie uma Arrow Function que receba um array de 10 números aleatórios e que retorne em ordem decrescente um novo array contendo apenas os números maiores que 10. (Utilizar o método filter)
Crie uma função que receba um array de 4 objetos representando produtos, cada um com nome e preço. A função deve retornar um objeto contendo o preço total de todos os produtos, o mais caro e o mais barato. (Utilizar o método reduce)
